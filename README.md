# login-page
# Tela de Login Responsiva

Este projeto é uma tela de login simples e responsiva, feita com **HTML**, **CSS** e **JavaScript**.  
Inclui ícone para mostrar/ocultar senha, validação básica e layout moderno.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Google Fonts (Inter)

## Funcionalidades
- Mostrar e esconder senha ao clicar no ícone.
- Layout responsivo para desktop e mobile.
- Campos com ícones dentro dos inputs.

## Como usar
1. Baixe o projeto ou clone o repositório:
2. https://github.com/LevyRibeiro111/login-page.git
3. Abra o arquivo `index.html` no navegador.
